Caso de Uso: Iniciar

Sumário: O jogador inicia uma partida

Ator Primário:  Jogador

Ator Secundário: s/ ator

Pré-condição:
	O oponente deve estar selecionado

Fluxo Principal:
	1 - O jogador inicia uma partida
	2 - O jogador efetua uma jogada

Fluxo Alternativo: 
	1.a - O jogador inicia uma partida
		1.a.1 - O sistema detecta a inicialização de uma partida
		1.a.2 - O sistema exibe uma mensagem de inicialização
		1.a.3 - O jogador efetua uma jogada

Fluxo Exceção:
	1.a - O jogador escolhe a opção errada
		1.a.1 - O jogador seleciona opção inválida
		1.a.2 - O sistema exibe uma mensagem de erro

Pós-condição:
	A partida é inicializada.
