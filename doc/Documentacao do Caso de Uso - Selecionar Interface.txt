Caso de Uso: Selencionar Interface

Sumário: O jogador selecionará a interface do jogo

Ator Primário:  Jogador

Ator Secundário: s/ ator

Pré-condição: s/ pré-condição

Fluxo Principal:
	1. - O jogador seleciona a interface do jogo
	2. - O jogador confirma a escolha

Fluxo Alternativo:
	1.a - O jogador seleciona a interface texto (linha de comando)
	1.b - O jogador seleciona a interface gráfica

Fluxo Exceção:
	1.a - O jogador escolhe opção errada
		1.a.1 - O jogador seleciona uma opção inválida
		1.a.2 - O sistema exibe uma mensagem de erro

Pós-condição:
	O tipo de interface é selecionado.
