Caso de Uso: Finaliza o Aplicativo

Sumário: O jogador finaliza o aplicativo do jogo

Ator Primário:  Jogador

Ator Secundário: s/ ator

Pré-condição:
	O aplicativo deve estar inicializado

Fluxo Principal:
	1. - O jogador finaliza o aplicativo
	2. - O jogador confirma a finalização do aplicativo

Fluxo Alternativo: 
	1.a - O jogador tenta finalizar o aplicativo durante uma partida
		1.a.1 - O sistema detecta a finalização durante uma partida
		1.a.2 - O sistema exibe uma mensagem de confirmação
	2.a - O sistema encerra o aplicativo
		2.a.1 - O sistema detecta a finalização do aplicativo
		2.a.2 - O sistema exibe uma mensagem de finalização

Fluxo Exceção: s/ fluxo de exceção

Pós-condição:
	O aplicativo é finalizado.
