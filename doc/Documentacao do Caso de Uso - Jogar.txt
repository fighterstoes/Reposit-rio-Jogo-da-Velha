Caso de Uso: Jogar

Sumário: O jogador realiza uma jogada

Ator Primário:  Jogador

Ator Secundário: s/ ator

Pré-condição:
	O oponente deve estar selecionado

Fluxo Principal:
	1. - O jogador escolhe a posição onde ele quer jogar
	2. - O jogador confirma a jogada

Fluxo Alternativo: 
	2.a - O jogo empata
		2.a.1 - O sistema verifica que não há mais jogadas a ser realizadas
		2.a.2 - O sistema exibe uma mensagem sobre o empate
		2.a.3 - O sistema inicia uma nova partida
	2.b - O jogador vence a partida
		2.b.1 - O sistema detecta que o jogador venceu
		2.b.2 - O sistema exibe uma mensagem sobre a vitória do jogador
		2.b.3 - O sistema inicia uma nova partida

Fluxo Exceção:
	1.a - O jogador escolhe uma opção inexistente
		1.a.1 - O jogador escolhe uma posição que não consta no tabuleiro
		1.a.2 - O sistema exibe uma mensagem de erro
	1.b - O jogador escolhe uma posição ocupada
		1.a.1 - O jogador escolhe uma posição já ocupada no tabuleiro
		1.a.2 - O sistema exibe uma mensagem de erro

Pós-condição:
	A jogada é realizada.
